# QuantAxis NextDay - 许愿池 (Wish Pool)

欢迎来到 QuantAxis NextDay 许愿池！这里是一个收集和管理量化交易相关功能需求和想法的地方。

Welcome to the QuantAxis NextDay Wish Pool! This is a place to collect and manage feature requests and ideas related to quantitative trading.

## 🌟 如何许愿 (How to Make a Wish)

### 通过 Issues 许愿 (Make a Wish via Issues)
1. 点击 [Issues](../../issues) 标签页
2. 点击 "New Issue" 创建新的许愿
3. 选择合适的模板填写您的许愿内容
4. 详细描述您希望实现的功能或改进

### 许愿分类 (Wish Categories)
- 🚀 **功能增强** (Feature Enhancement) - 新功能请求
- 🐛 **问题修复** (Bug Fix) - 发现的问题和修复建议  
- 📈 **策略相关** (Strategy Related) - 交易策略相关需求
- 🔧 **工具改进** (Tool Improvement) - 现有工具的改进建议
- 📚 **文档完善** (Documentation) - 文档相关的改进
- 💡 **创意想法** (Creative Ideas) - 其他创新想法

## 📋 许愿模板 (Wish Template)

```markdown
### 许愿标题 (Wish Title)
简明扼要地描述您的需求

### 详细描述 (Detailed Description)
详细说明您希望实现的功能或改进

### 使用场景 (Use Case)
描述在什么情况下会用到这个功能

### 期望效果 (Expected Outcome)
描述实现后的预期效果

### 优先级 (Priority)
- [ ] 高 (High)
- [ ] 中 (Medium)  
- [ ] 低 (Low)

### 附加信息 (Additional Information)
任何其他相关信息或参考资料
```

## 🤝 贡献指南 (Contributing Guidelines)

1. **明确性**: 请尽可能详细地描述您的需求
2. **唯一性**: 提交前请搜索是否已有相似的许愿
3. **建设性**: 提供具体的改进建议而非单纯的抱怨
4. **耐心**: 许愿的实现需要时间，请保持耐心
